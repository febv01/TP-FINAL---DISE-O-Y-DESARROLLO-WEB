Sobre Nosotros

Somos un equipo apasionado por el desarrollo web y la creación de experiencias digitales. Este proyecto es una demostración de un carrusel de imágenes interactivo, desarrollado utilizando HTML, CSS y JavaScript puro.

Nuestro objetivo es aprender y aplicar los conceptos fundamentales de la programación web para crear interfaces visualmente atractivas y funcionales.

Nuestra Misión

Brindar soluciones web innovadoras que inspiren y conecten a las personas a través de la tecnología.
