¿Qué es el Testing Funcional?

El Testing Funcional se centra en verificar que cada función del software opere conforme a los requisitos especificados. Se basa en pruebas con entradas y salidas específicas.

Ejemplos de Testing Funcional

  • Verificar que el formulario de registro valida correctamente los campos obligatorios.
  • Probar el flujo de compra en un sitio de comercio electrónico.
  • Evaluar el correcto funcionamiento de un buscador interno.

Herramientas para Testing Funcional

  • Selenium
  • Katalon Studio
  • Cucumber
